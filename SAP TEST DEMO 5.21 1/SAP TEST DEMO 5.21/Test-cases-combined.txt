
# SAP test 1

BP_IT_B2P_GUI_Report Asset HistorySheet. (Row 2/3)
Calling up and evaluating the report data

自然语言：
1.进入 Menu， 点击setting，点击visualization 并启用show Ok code field, 并保存。
2.在 矩形输入框 输入 S_ALR_87011990 ，点击右下角execute
3. company code输入8540， Asset number 输入1010001732
4. 点击跳出栏目的第一个。 
5.Report date 输入上个月的最后一个自然日, 并勾选List assets, 执行Execute 查询。
6.在查询到的Asset History Sheet页面中，滑动到最右边，查找Curr.bk.val.列的值，记录为变量A1.
7. 返回 SAP 主页, 选择Favorites文件夹下面的Asset Balances, 执行Execute。
8.company code处输入8540, Asset number输入1010001732。
9. 点击跳出栏目的第一个。  
10. Report date输入30.04.2026, 勾选List assets, 执行Execute。 
11. 记录查询到的数据表格 第一行，Book val.l列的值为变量A2.
12. A1和A2相等则测试用例执行成功。

JS：
async function run(agent) {
  await agent.aiTap("进入 Menu");
  await agent.aiTap("点击setting");
  await agent.aiTap("点击visualization");
  await agent.aiTap("启用show Ok code field");
  await agent.aiTap("保存");
  await agent.aiInput("矩形", { value: "S_ALR_87011990" });
  await agent.aiTap("点击右下角execute");
  await agent.aiInput("company code", { value: "8540" });
  await agent.aiInput("Asset number", { value: "1010001732" });
  await agent.aiTap("点击跳出栏目的第一个");
  await agent.aiInput("Report date", { value: "30.04.2026" });
  await agent.aiTap("勾选 List assets 复选框");
  await agent.aiTap("点击 Execute 按钮执行查询");
  await agent.aiTap("在查询到的Asset History Sheet页面中");
  await agent.aiAct("按住底部的横向滚动条的滑块，拖到最右端");
  const result = await agent.aiQuery("string, 从查询结果表格中读取 Curr.bk.val.列，保存为变量 A1");
  await agent.aiTap("返回 SAP 主页");
  await agent.aiTap("展开 Favorites 文件夹并选择 Asset Balances");
  await agent.aiInput("company code", { value: "8540" });
  await agent.aiInput("Asset number", { value: "1010001732" });
  await agent.aiTap("点击跳出栏目的第一个");
  await agent.aiInput("Report date", { value: "30.04.2026" });
  await agent.aiTap("勾选 List assets 复选框");
  await agent.aiTap("点击 Execute 按钮执行查询");
  const result = await agent.aiQuery("string, 记录查询到的数据表格 第一行");
  const result = await agent.aiQuery("string, Book val.l列的值为变量A2.");
  await agent.aiAssert("如果 A1 和 A2 相等，则判定测试用例执行成功；否则判定失败");
}
