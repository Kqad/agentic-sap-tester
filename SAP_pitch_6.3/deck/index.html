Pitch · §09 / command center
Command center · live during every run

The whole fleet, on one panel. Every number derivable.

  • Total · Passed · Failed · Running · Pending — raw counts across the case library. Updated the moment a run starts, finishes, or is queued.
  • Pass rate — passed ÷ (passed + failed). Excludes never-run cases so seeding new ones doesn't depress the number. Sparkline tracks the rolling window of recent runs.
  • Cache hit — observed cached-step responses ÷ observed total steps in the current session. Each cached step replays a recorded xpath instead of re-asking the model.
  • 7-day activity — runs per local day for the trailing week, with today's bar highlighted.
  • Avg duration — mean wall-clock duration across all completed runs.
  • Time saved — (cache-enabled passed runs × ~27 steps × 75% typical hit rate × 4 s / hit) + (live-observed hits × 4 s). Click for the line-by-line derivation.
  • Fleet health — weighted composite: pass rate (60%) + cache efficiency (25%) + recency of last run (15%). Green ≥ 80, amber 55–79, red below.

Pitch · §19 / recovery · architecture
Recovery architecture · graceful degradation

Stuck? The platform escalates — never panics.

Every recovery starts with the smallest effective action. If that's not enough, the platform reaches one level deeper, and only then one level deeper again. Cache makes each escalation cheap — replayed steps cost milliseconds, not LLM calls. Three tiers, in order, then the run continues.

!
Entry
STUCK
The runner is blocked.
Locator gone, popup in the way, screen drifted.
1Light · minimal action
Close the popup.
Try the cache.
▶ Is something blocking the locator?
Dismiss any modal / dialog, wait a beat, then let the runner retry the same step from cache. If it hits, done.
cost ~10 scovers blockers
2Middle · partial rollback
Auto-judge depth.
Cache replays back.
▶ Can a small rollback unstick it?
Platform auto-judges the rollback depth (typically 1–3 prior steps); cache replays them in seconds. Then retry the stuck step. If it hits, done.
cost ~15 scovers state drift
3Heavy · full reset
Click SAP logo.
Replay from step 1.
▶ Nothing small works. Reset to home.
Agent clicks the SAP logo to land on home, then the runner replays every prior step from cache and retries the stuck step.
cost ~30 scovers anything
✓
Exit
PASSED
The stuck step succeeds.
The run finishes — without you.
Try the lightest fix first · Escalate only when the previous tier didn't unstick it · The cache makes every escalation cheap
